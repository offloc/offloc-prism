Prism
=====

A URL Shortener and Custom Header Injector.

This project contains the core components of Prism that are independent
from any front-end implementation.


Requirements
------------

 * PHP 5.3+


Installation
------------
 
Through [Composer][1] as [offloc/prism][2].
